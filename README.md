# Developer-AWS
Laboratórios de developer da AWS:
1 ° Laboratório – Explorando a AWS com o Amazon EC2 - Esse laboratório irá te fornecer uma visão básica e prática do que é uma instância EC2, como inicializá-la pelo console e via CloudShell.
2° Lboratório - Criando um Orçamento na AWS com Budget - Este laboratório orienta passo a passo como configurar um orçamento de gastos na AWS utilizando o serviço AWS Budgets. A atividade inclui a criação do orçamento, configuração de alertas por e-mail e verificação das definições aplicadas.
